## 2. Title

**Redux Shopping Cart – State Management using Redux Toolkit**

---

## 3. Approach

This experiment implements a shopping cart application using React and Redux Toolkit for efficient state management.

* A Redux store is configured using `configureStore`.
* A cart slice is created to manage cart state including items, quantity updates, and removal operations.
* React components are connected to the Redux store using `useSelector` and `useDispatch` hooks.
* Users can update product quantities dynamically and remove items from the cart.
* The total price is calculated automatically based on the current state.
* The application follows a modular structure with separation of concerns (store, slice, components).
* The project is built and deployed using Netlify by generating optimized static files.
